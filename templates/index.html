기능 소개

제품 리스트 추가 - 메인 페이지의 DB 테이블에 제품을 새로 등록

미리보기에 등록 - 선택한 데이터를 미리보기 DB에 제출

미리보기 - 최종화면의 데이터와 가장 마지막으로 "미리보기에 등록"한 1개의 데이터를 모두 미리 보여줌

최종화면에 등록 - 미리보기 안에 있는 버튼으로서, 마지막으로 "미리보기에 등록"한 데이터("대기"에 해당)를 최종화면 DB에 등록

최종화면 - 최종적으로 내보낸 데이터들을 텍스트 형식으로 새 창에 보여줌

이용 순서

제품 리스트 추가 → 체크박스 클릭 → 미리보기에 등록 → 미리보기 → 최종화면에 등록 → 최종화면

유의 사항

"미리보기에 등록"과 "최종화면에 등록"는 1개씩만 가능

아무것도 선택하지 않고 "미리보기에 등록"를 누를 경우 완료 문구는 뜨지만, 실제로 미리보기에 등록 되지는 않음

미리보기 안에 "대기"에 해당하는 데이터는 가장 최근에 "미리보기에 등록"한 항목임

┗> 대기를 "최종화면에 등록" 하더라도 "대기"는 바뀌지 않음 ( 새로 다른 것을 "미리보기에 등록"해야 바뀜 )

최종화면의 데이터를 삭제하고 싶으면 미리보기에서 삭제하면 됨 ( 미리보기에서도 삭제됨 )

사이트 이용 종료 시 보안을 위해 "로그아웃" 클릭하기 ( 관리자 모드였더라도, "Admin 로그아웃"은 저절로 실행됨 )
