★ 기능 소개

▶ 1. 추가하기 - 메인 페이지의 DB 테이블에 제품을 새로 등록

▶ 2. 대기하기 - 선택한 데이터를 미리보기 DB에 제출

▶ 3. 미리보기 - 최종화면의 데이터와 가장 마지막으로 대기하기한 1개의 데이터를 모두 보여줌

▶ 4. 내보내기 - 미리보기 안에 있는 내보내기 버튼으로서, 마지막으로 대기하기한 데이터를 최종화면 DB에 등록

▶ 5. 최종화면 - 최종적으로 내보낸 데이터들을 텍스트 형식으로 새 창에 보여줌

★ 이용 순서

▶ 1.추가하기(생략가능) → 2.제품 선택 후 대기하기 → 3.미리보기 → 4.내보내기 → 5.최종화면

★ 유의 사항

▶ 선택은 1개씩만 가능

▶ 미리보기 안에 대기에 해당하는 데이터는 가장 최근에 "대기하기"한 항목임

▶ ┗> 대기를 내보내기 하더라도 대기는 바뀌지 않음(새로 다른 것을 "대기하기"해야 바뀜)
